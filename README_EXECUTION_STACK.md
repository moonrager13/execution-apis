# Execution Stack Upgrade

## Goals
- Ethereum + Base deployment support
- Smart contract interaction tooling
- Automated testing and verification
- Secure owner-controlled execution flows

## Deployment workflow
1. Configure environment variables locally.
2. Deploy to testnet.
3. Verify source code.
4. Review permissions.
5. Deploy to mainnet only after testing.

## Networks
- Base
- Ethereum

## Security requirements
- Use hardware wallet or secure signer.
- Do not commit secrets.
- Add access controls and event logging.
