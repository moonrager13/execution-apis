# Build Status

Current stack:

- Node.js 22
- Hardhat deployment tooling
- Base network configuration
- Ethereum mainnet configuration
- GitHub Actions CI
- GitHub Secrets deployment workflow
- Agent execution architecture
- Contract verification workflow

Deployment workflow:

1. Add repository secrets.
2. Run the Deploy Contracts GitHub Action.
3. Select Base or Ethereum mainnet.
4. Verify the deployed contract.

Production deployments require the deployer wallet approval and funded network account.
